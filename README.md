## ⚔ OVERVIEW

Virus Scanner Pro, VirusTotal API v3 kullanan gelişmiş bir masaüstü güvenlik analiz platformudur.

Dosyaların SHA256, SHA1 ve MD5 hashlerini hesaplayarak VirusTotal üzerinde sorgular, 70+ antivirüs motorundan gelen sonuçları tek ekranda toplar ve kullanıcıya detaylı analiz raporu sunar.
